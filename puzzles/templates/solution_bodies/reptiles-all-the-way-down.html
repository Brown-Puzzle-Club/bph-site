Once they start realizing that not all images are reverse searchable, and not all text clues are specific to one species, that’s when they really start the puzzle. This is a game of snakes and ladders. The ladders are given, and the snakes can be determined from sample gameplay. This is the completed board: Each snake is made up of three clues: two text and one image. This is the case for every clue. Now it is time to identify what reptile each snake is clueing. The flavor text says to line up the heads, which is cluing the order that you’ll need them in for extraction (not necessary at this point but keeps it organized).
