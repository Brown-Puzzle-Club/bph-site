The next step is to realize that each color refers to a different keyboard layout. This is the big aha moment. This means that one word written on one keyboard layout will equal a different word when written on a different keyboard layout. Doing some quick research on different common keyboard layouts and some trial and error with the different words, you should be able to figure out which color corresponds to which keyboard. For example, “lamb” when written on the “QWERTY” layout becomes “iamb” when the same exact keys are pressed on the “Colemak” layout. Here are the keyboard layouts used:
Red QWERTY
Blue COLEMAK
Green DVORAK
Black ???
Now you are probably wondering how the black words fit it, since they don’t match any existing keyboard layout. Instead of corresponding to a keyboard that already exists, the black words form an entirely new layout that you must create to solve the puzzle. For example “musk = send” is telling you that “s” on this new keyboard will be in the same place that “m” is on the Colemak keyboard. Once you transcribe all the black words onto a new keyboard layout, this is what you get: This new keyboard pretty clearly contains the name ALICE MUNRO, which is the answer to this puzzle! {% endblock %}
