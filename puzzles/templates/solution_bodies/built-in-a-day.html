{% extends "solution.html" %} {% block puzzle-author %} Thomas Gordon {% endblock %} {% block solution-body-md %} As the flavortext says, this is a skyscrapers logic puzzle, with some weird looking clues and numbers in the puzzle. Our first aha is to realize how this skyscraper variant functions: in addition to giving the number of skyscrapers visible, each clue also gives letters which are 'on top' of those visible skyscrapers. All of these given words are valid Latin words with the numbers being the correct enumeration (word length). This can be figured out from the theming, the flavortext or some specific clues of the puzzle. Another specific giveaway that these are Latin words is the specification of gender, case, and tense in some places. Our next port of call might be to answer the clues:

Note that some of these words may be not uniquely known; we use checks from the grid to disambiguate between different words. Similarly, we can take these numbers as clues in a skyscraper puzzle, and solve it to get these 4 grids. However, notably, the numbers alone do not lead to unique skyscraper solutions. The solver must also pay attention to the letters which each observer "sees" on top of each skyscraper (or 'pillar') in order to solve the grids uniquely. Letters are placed such that if a spectator reads from the highest number to the lowest number (in a "top to bottom" grid) or the lowest number to the highest number (in a "bottom to top" grid), then those observed letters will spell the Latin words clued at that position. Not every pillar will have a letter: only those seen by the clues. With this aha known, each grid is now unique for both Latin clues and Skyscraper numbers. The solutions are below: Grid 1
